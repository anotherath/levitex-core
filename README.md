# Levitex Core

Levitex is a decentralized exchange (DEX) focused on providing a smooth, floating, and dynamic Web3 trading experience.

## Features

- **Next.js & React**: Built with the latest performance optimizations.
- **Modern UI**: Futuristing design with glassmorphism and subtle animations.
- **Three.js Integrations**: Floating geometric backgrounds for an immersive experience.
- **Responsive Design**: Tailored for both desktop and mobile views.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js 15+
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, Three.js
- **Icons**: Lucide React
